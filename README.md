# 交个朋友吧！

一个有趣的交互式网页，包含可爱的动漫头像和有趣的按钮交互效果。

## 功能特点

- 可爱的动漫头像
- 动态的按钮交互
- 点击"不行"按钮时的有趣反应
- 点击"可以"按钮后的成功页面

## 技术栈

- HTML5
- CSS3
- JavaScript

## 在线预览

访问 [https://你的用户名.github.io/仓库名](https://你的用户名.github.io/仓库名) 查看在线演示。

## 如何使用

1. 确保你有以下文件：
   - index.html
   - avatar.png (动漫头像图片)

2. 直接在浏览器中打开 index.html 即可使用

## 技术特点

- 纯HTML/CSS/JavaScript实现
- 无需额外依赖
- 移动端友好的响应式设计

## 自定义

你可以通过修改以下内容来自定义网站：

- 替换 avatar.png 来更换头像
- 修改 CSS 中的颜色值来改变主题色
- 在 JavaScript 中修改交互文本
